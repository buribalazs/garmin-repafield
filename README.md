# garmin-repafield
Trail running focused Garmin watch DataField (for myself)

# Data shown

- heart rate
  - average
  - current
  - max
  - histogram
- pace
  - current
  - average
- distance
- elevation
  - current
  - gain
  - loss
- cadence
- distance to destination gauge
